---

This page walks you through everything you need to get your own Matchmaker bot running — from creating a Discord application to starting your first match in a server channel.

<Steps>
  <Step title="Create a Discord bot application">
    Go to [discord.com/developers/applications](https://discord.com/developers/applications) and create a new application.

    1. Click **New Application**, give it a name, and confirm.
    2. Open the **Bot** tab in the left sidebar.
    3. Click **Add Bot** to create a bot user for your application.
    4. Under **Token**, click **Reset Token** and copy the value — you'll need it in step 4.
    5. Scroll down to **Privileged Gateway Intents** and enable **Server Members Intent**.
    6. Save your changes.

    <Note>
      Matchmaker requires the `members` intent to look up guild members when building match lobbies. Without it, the bot will start but player lookups will fail.
    </Note>
  </Step>

  <Step title="Clone the repository">
    Clone the Matchmaker source code to your local machine:

    ```bash
    git clone https://github.com/seanchencs/matchmaker.git
    cd matchmaker
    ```
  </Step>

  <Step title="Install Poetry and dependencies">
    Matchmaker uses [Poetry](https://python-poetry.org/docs/) for dependency management. Install it if you don't have it already, then install the project dependencies:

    ```bash
    poetry install
    ```

    This creates a virtual environment and installs all required packages, including `py-cord`, `trueskill`, and `python-dotenv`.
  </Step>

  <Step title="Set the TOKEN environment variable">
    The bot authenticates with Discord using the token you copied in step 1. You can provide it as a shell environment variable or via a `.env` file.

    <CodeGroup>

    ```bash Environment variable
    export TOKEN=your_bot_token_here
    ```

    ```ini .env file
    TOKEN=your_bot_token_here
    ```

    </CodeGroup>

    If a `.env` file is present in the project root, Matchmaker loads it automatically on startup.
  </Step>

  <Step title="Run the bot">
    Start the bot with Poetry:

    ```bash
    poetry run python main.py
    ```

    When the bot is ready, you'll see `Logged in as <your bot name>` printed to the terminal. Detailed logs are written to `discord.log` in the project directory.
  </Step>
</Steps>

## Running your first match

Once the bot is online and added to your server, go to any text channel and run the `/start` command. This opens a match lobby with **Join** and **Leave** buttons that players can click to sign up.

When you have enough players, click **Start Game** to let Matchmaker divide them into balanced teams based on their current TrueSkill ratings. After the match, click **Record Result** to submit scores and update everyone's MMR.

<Tip>
  Prefer a containerized setup? You can run Matchmaker with Docker instead of installing Python and Poetry directly. See the [Docker deployment guide](/deployment/docker) for instructions.
</Tip>
